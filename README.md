PrajaNetra AI
PrajaNetra AI is a multilingual civic intelligence platform designed to modernize public grievance management.

Features
Multilingual chatbot (Telugu, Hindi, English)
Voice complaint registration
Manual complaint form
AI-based complaint classification
Location-based priority detection
Government officer dashboards
Civic heatmap visualization
Predictive civic intelligence
Tech Stack
Frontend: React / Next.js
Backend: FastAPI / Flask
Database: Firebase Firestore
AI: NLP-based complaint analysis
Maps: Leaflet / Mapbox

Prototype
This repository contains the prototype implementation for the PrajaNetra AI civic governance platform.
